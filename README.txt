How to add photos to the Birthday Website:

1. Opens the "images" folder in this directory.
2. Paste your photos into that folder.
3. Rename your photos to numbers: 
   - 1.jpg or 1.png
   - 2.jpg or 2.png
   - 3.jpg or 3.png
   - ... up to 10.jpg

How to add your SONG:
1. Copy your song file (mp3) into this main project folder.
2. Rename it to "song.mp3".

Note: The specific file names are important for the website to automatically find them! 

Enjoy the surprise!
